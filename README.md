# Solid/UI

This project aims to port the following components to SolidJS:

- [shadcn/ui](https://ui.shadcn.com)
- [cmdk](https://github.com/pacocoursey/cmdk)
- [tremor](https://www.tremor.so/)

![screenshot](public/screenshot.png)
