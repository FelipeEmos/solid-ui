# solid-shadcn/ui

**What is this?**

This is a port of the epic components built by @shadcn which you can find [here](https://ui.shadcn.com).

![screenshot](screenshot.jpg)

- Button
- Card
- Checkbox
- Dialog
- Input
- Label
- Select
- Switch
- Table
- Tabs
- Textarea
